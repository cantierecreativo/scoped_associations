# DragonflyImage

DragonflyImage lets you associate any model with multiple images

## Installation

Add this line to your application's Gemfile:

    gem 'dragonfly_image'

And then execute:

    $ bundle

Or install it yourself as:

    $ gem install dragonfly_image
